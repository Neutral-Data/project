Uploading Local File

Step 1: Navigate to the file upload view

You can access the view from this link.

Step 2: Select a file

Choose one of your local files with a .csv or .xlsx extension.

Step 3: Check that it is the file you want

The first 100 lines of the file will be previewed so that you can observe it and not make mistakes.

Step 4: Choose the type of analysis

You can choose, parse the headers, analyze the columns or rows, check the profanity, enter your own terms or any set of these options.

Step 5: Upload the query

Click on the upload button and then on the continue button.

Step 6: Wait

The analysis will start, please keep in mind that large files require a lot of computing power, so it will take a while.

Step 7: View the results

The analysis results will be displayed, you can take the time you need to review it.

*Step 8: Customize Settings

Optionally, you can download the file with the same input format. This file is limited to one of the options as long as they are individual and not Profanity (if more options are mixed, it will not work correctly).

Choose File GitHub Repository

Step 1: Navigate to the github repositories view

You can access the view from this link.

Step 2: Enter your GitHub username

Please note that you will only be able to see your public repositories.

Step 3: Choose a file
